### 중요 사항
- **여러 줄에 걸친 색상 마커는 지원하지 않음**
- MD 파일 작성 시 색상 마커는 반드시 한 줄 안에 작성:
  ```markdown
  ✅ 올바른 예: {{green:초록색 텍스트입니다}}
  ❌ 잘못된 예: {{green:
  여러 줄에 걸친
  텍스트}}
  ```

### 표 스타일
- 표 헤더: CENTER 정렬
- 표 데이터: JUSTIFY 정렬
- 모든 표 셀: 왼쪽 여백 0, 첫줄 들여쓰기 0
