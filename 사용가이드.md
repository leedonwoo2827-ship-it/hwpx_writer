# HWPX Writer 사용 가이드

## 📚 개요
마크다운 파일을 한글(HWPX) 문서로 변환하는 MCP 서버입니다.

## 🚀 빠른 시작 (Claude Desktop)

### 1. MCP 서버 등록
Claude Desktop의 MCP 설정에 hwpx-writer 서버를 등록합니다.

### 2. 마크다운 파일 작성
제안서를 마크다운 형식으로 작성합니다.

**색상 마커 사용 예시**:
```markdown
# 제안서 제목

## {{green:초록색 섹션 제목}}

일반 텍스트입니다. {{red:빨간색 강조}} 텍스트도 가능합니다.

| 항목 | {{green:내용}} |
|------|----------------|
| A | {{red:중요 데이터}} |
```

**⚠️ 주의: 색상 마커는 반드시 한 줄 안에 작성**
```markdown
✅ 올바른 예: {{green:초록색 텍스트}}

❌ 잘못된 예:
{{green:
여러 줄에 걸친
텍스트
}}
```

### 3. HWPX 변환
Claude Desktop에서 다음과 같이 요청:

```
hwpx-writer를 사용해서
C:\Users\...\proposal-body.md 파일을
HWPX로 변환해줘.
```

## 📝 마크다운 문법

### 헤딩 레벨
```markdown
# 섹션 제목 (level 1)
## 중간 제목 (subtitle level 1)
### 소제목 (subtitle level 2)
#### 항목 제목 (level 4, ○)
##### 세부 항목 (level 5, ―)
###### 참고 사항 (level 6, ※)
```

### 목록
```markdown
- 본문 항목 (□)
  - 세부 항목 (○)
    - 보충 설명 (―)
      - 참고/주석 (※)
```

### 표
```markdown
| 헤더1 | 헤더2 |
|-------|-------|
| 데이터1 | 데이터2 |
```

**표 스타일**:
- 헤더: 가운데 정렬, 회색 배경
- 데이터: 양쪽 정렬
- 모든 셀: 왼쪽 여백 0, 들여쓰기 0

### 이미지
```markdown
![이미지 설명](images/파일명.png)
```

### 색상 마커
```markdown
{{red:빨간색}}
{{green:초록색}}
{{blue:파란색}}
{{yellow:노란색}}
```

## 🎨 스타일 커스터마이징

`proposal-styles.json` 파일을 수정하여 스타일을 변경할 수 있습니다.

```json
{
  "styles": {
    "level2": {
      "font": "Noto Serif KR",
      "size": 10,
      "leftMargin": 0,
      "align": "justify"
    },
    "table": {
      "font": "Noto Sans KR",
      "size": 9
    }
  },
  "colors": {
    "red": "#dc2626",
    "green": "#16a34a"
  }
}
```

## 🔧 문제 해결

### Q1. 색상 마커가 그대로 표시됨
**원인**: 여러 줄에 걸친 색상 마커 사용

**해결**:
```markdown
❌ 잘못된 사용:
{{green:
여러 줄
텍스트
}}

✅ 올바른 사용:
{{green:한 줄로 작성한 텍스트}}
```

### Q2. 표 셀 들여쓰기가 많이 적용됨
**원인**: 이전 버전의 hwpx_generator 사용

**해결**: 최신 버전 (2026-03-19) 사용 확인

### Q3. 이미지가 문서 끝에 몰림
**원인**: `image_paths` 파라미터 사용

**해결**: 마크다운 본문에 `![](images/파일명.png)` 태그로 삽입
